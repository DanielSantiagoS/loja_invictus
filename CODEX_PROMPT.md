# Prompt para usar no Codex

Crie um aplicativo Flutter/Dart chamado "Loja Online Simples", para Android Studio, seguindo uma metodologia didática passo a passo. O app deve ter:

1. Página Inicial com AppBar azul, botão "Ver Produtos" e botão "Carrinho".
2. Página de Produtos com cards contendo imagem/ícone, nome, preço, descrição curta, estoque e botão "Selecionar".
3. Página de Detalhes com ID do produto, imagem/ícone maior, preço, estoque, descrição longa, botão "Adicionar ao Carrinho" e botão "Ver Mais Produtos".
4. Carrinho de Compras com ID, nome, preço unitário, quantidade, subtotal por item, frete, impostos e total.
5. Controle de quantidade com botões + e - e validação para impedir quantidade acima do estoque.
6. Botão "Cancelar Pedido" que limpa o carrinho.
7. Tela de Finalização com endereço de cobrança, endereço de entrega, resumo do pedido e número de confirmação.
8. Inventário externo em `assets/products.json`.

Use Flutter Material Design, código comentado e linguagem clara para estudantes iniciantes do ensino médio técnico.
